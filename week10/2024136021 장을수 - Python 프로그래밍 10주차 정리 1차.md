# 9장 리스트와 딕셔너리
## 자료형
리스트: `myList = [ 10, 20, 30, 40, 50 ]`
튜플(값 변경 불가): `myTuple = ( 10, 20, 30, 40, 50 )`
딕셔너리:  `myDict = { 1: "one", 2:"two", 3:"three" }`
세트(중복 불가): `mySet = { "one", "two", "three" }` 

## 리스트 선언
1. 리스트의 참조값을 변수에 저장한다.
2. 대괄호로 데이터를 묶는다.
3. 리스트를 변수에 저장한다.
예시: `리스트_참조값 = [ "변수1", "변수2", "변수3"]`

## 리스트 항목 접근하기
```python
>>> letters = ['A', 'B', 'C', 'D', 'E', 'F']

>>> letters[0]
A
>>> letters[1]
B
>>> letters[2]
C
>>> letters
Traceback (most recent call last):

letters[6]

```

## 리스트 길이
리스트에 몇 개 들었는지
len() 함수
예: `len(리스트_참조값)`

## in 연산자
어떤 값이 리스트 내부에 있는 지를 확인
numbers 리스트 체크: 6 in numbers
단어 체크: `'e' in 'Hello'`
- 예: 
    ```python
    >>> numbers = [ 1, 2, 3, 4, 5, 6 ]
    >>> 6 in numbers
    True
    
    >>> numbers = [ 1, 2, 3, 4, 5, 6 ]
    >>> 10 not in numbers
    True
    
    >>> 'e' in 'Hello'
    True
    
    ```
    

## 리스트에 저장된 데이터를 하나씩 꺼내기
```python
myList = [ "우유", "사과", "두부", "소고기"]

for item in myList :
    print(item)
```

```python
myList = [ "우유", "사과", "두부", "소고기"]

for i in range(len(myList)) :
    print(myList[i])
```
출력: 
우유
사과
두부
소고기

## 리스트 항목 교체
```python
myList = [ "우유", "사과", "두부", "소고기"]
myList[1] = '커피’

print(myList)
```
출력: ['우유', '커피', '두부', '소고기']

## 공백 리스트만들고 값 추가하기
공백 리스트 생성 가능, `append()`로 값 추가
- 예
    ```python
    myList = [ ]
    myList.append("우유")
    myList.append("사과")
    myList.append("두부")
    myList.append("소고기")
    print(myList)
    
    ```
리스트 중간에 새로운 값 입력하기: `insert()`

## 점의 의미
변수와 함수를 묶은 것 (예: 객체.메소드())

## 항목 삭제하기
항목 삭제: `remove()`
특정 인덱스 항목 삭제: `pop(index값)`

## 리스트 탐색하기
```python
myList = [ "우유", "사과", "두부", "소고기"]
if "소고기" in myList:
    i = myList.index("소고기")         # i는 3
```

## 리스트 합치기
myList와 yourList 합치기거나(`myList+yourList`)
배율로 늘리는 것도 가능함(`myList*2`)

## **슬라이싱(slicing)**
리스트에서 한 번에 여러 개의 항목을 추출하는 기법이다.
`letters[**~에서_부터**:**~까지**]`, 비어있으면 전체

## 내장 함수
`min()` - 리스트에서 최소값
`max()` - 리스트에서 최대값
`sum()` - 리스트 안에 값들의 합계

## 리스트 정렬
`리스트_참조값.sort()` 또는 `sorted(리스트_참조값)`
기본적으로 0->9순, A->Z순으로 출려됨
반대로 하려면 `sorted(리스트_참조값, **reverse=True**)`
